Installation Instructions

Installation is fairly simple.

  1. Download the Executable Jar File above.
  2. Place the jar where you want to install (Keep in mind that it downloads files to its location)
  3. Run the Jar
  4. Sit and watch it do all the work for you.

Help!!!!!

If you can not get it to run, you most likely do not have java installed (at least not the right version). It requires Java 8.

If you have Java 8, and it still won't work, try opening command prompt (click Windows key + r [or start->run] and enter "cmd"), and type the following (Note that [ENTER] means press the Enter key:

cd [where your file is located] [ENTER]
java -jar PhysiologyQuiz.jar [ENTER]

For example, if you saved to your desktop and are running windows:

cd "C:\Users\[Username]\Desktop" [ENTER]
java -jar PhysiologyQuiz.jar [ENTER]

If that doesn't work, then there is most likely something wrong on my end, so just tell me, and I will look into it.
